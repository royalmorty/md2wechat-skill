# 视觉设计示例

## 示例1：封面图设计

### 文章主题：春茶选购指南
### 封面图提示词
```
为"春茶选购指南"生成微信公众号封面图，要求：
- 尺寸：2560x1440像素，16:9比例
- 主题：春茶、茶叶、选购
- 风格：温暖治愈，春日清新
- 主色调：嫩绿色、暖白色
- 元素：新鲜茶叶、茶具、春日阳光
- 构图：茶叶特写为主体，背景虚化茶园
- 光线：自然光，温暖明亮
- 文字："春茶选购指南"，字体优雅
```

### 设计说明
- **视觉焦点**：新鲜茶叶的嫩芽，突出春茶的鲜嫩
- **色彩心理**：绿色代表自然、健康，暖色调营造温馨感
- **构图技巧**：使用浅景深，突出主体，背景虚化增加层次感
- **文字设计**：选择优雅的无衬线字体，颜色与整体协调

## 示例2：插图设计

### 文章主题：绿茶冲泡技巧
### 插图提示词
```
为"绿茶冲泡技巧"生成步骤说明插图，要求：
- 尺寸：800x600像素，适合手机屏幕
- 主题：绿茶冲泡过程
- 风格：简洁明了，清新自然
- 配色：淡绿色、白色、木色
- 元素：玻璃杯、绿茶、热水、温度计
- 布局：步骤式排列，1-2-3-4-5
- 风格：扁平化插画风格
```

### 设计说明
- **信息传达**：每个步骤配简单图标，直观易懂
- **色彩统一**：使用统一的绿色系，保持视觉一致性
- **版式设计**：步骤清晰排列，方便阅读理解
- **适配考虑**：尺寸适合手机观看，不会占用过多流量

## 示例3：对比图设计

### 文章主题：真假茶叶鉴别
### 对比图提示词
```
为"真假茶叶鉴别"生成对比图，要求：
- 尺寸：1200x800像素，横向对比
- 主题：真茶vs假茶的对比
- 风格：简洁专业，权威可信
- 配色：左侧绿色系（真茶），右侧红色系（假茶）
- 元素：茶叶外形、茶汤颜色、叶底状态
- 布局：左右对比，中间分割线
- 标注：关键区别点用箭头指示
```

### 设计说明
- **对比效果**：使用颜色区分，绿色代表正确，红色代表错误
- **信息层次**：主要对比点突出显示，次要信息弱化处理
- **专业感**：整体风格偏向专业权威，增加可信度
- **易读性**：关键信息用图标和简短文字标注

## 示例4：信息图设计

### 文章主题：2024春茶市场数据
### 信息图提示词
```
为"2024春茶市场数据"生成信息图，要求：
- 尺寸：1000x1400像素，竖版
- 主题：春茶市场数据分析
- 风格：现代简约，数据可视化
- 配色：绿色系为主，橙色点缀
- 图表：饼图、柱状图、折线图
- 数据：价格趋势、产区分布、品质等级
- 布局：从上到下的信息层级
```

### 设计说明
- **数据可视化**：将复杂数据转化为直观的图表
- **色彩编码**：不同数据类型使用不同颜色区分
- **信息层级**：重要数据突出显示，次要数据简化处理
- **品牌一致性**：保持与文章整体风格一致

## 示例5：故事场景图

### 文章主题：茶农的一天
### 场景图提示词
```
为"茶农的一天"生成故事场景图，要求：
- 尺寸：1200x800像素，横向构图
- 主题：茶农采茶的日常生活
- 风格：温暖写实，生活气息
- 时间：清晨日出时分
- 地点：高山茶园
- 人物：茶农正在采茶
- 氛围：宁静祥和，自然美好
- 光线：温暖的晨光
```

### 设计说明
- **情感共鸣**：通过温暖的画面唤起读者对自然生活的向往
- **故事性**：画面具有叙述性，能够支撑文章内容
- **真实性**：写实风格增加文章的可信度和亲和力
- **美学价值**：在保证信息传达的同时注重美感

## 设计原则总结

### 1. 内容相关性
- 图片必须紧扣文章主题
- 避免使用与内容无关的装饰性图片
- 每个图片都应该有明确的信息传达目的

### 2. 视觉一致性
- 保持整体配色方案的统一
- 字体风格保持一致
- 图标和图形元素风格统一

### 3. 平台适配性
- 考虑微信公众号的特殊显示环境
- 图片尺寸适合手机观看
- 文件大小控制在合理范围内

### 4. 用户体验
- 信息传达清晰直观
- 避免过于复杂的设计
- 考虑不同用户群体的接受度
