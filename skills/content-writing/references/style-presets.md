# 写作风格预设详解

本文档详细介绍 writer 系统支持的三种写作风格预设。

## 风格概览

| 风格 | 特点 | 适用场景 | 难度 |
|-----|------|---------|------|
| dan-koe | 简洁有力，金句频出 | 个人成长、商业洞察 | ⭐⭐⭐ |
| cultural-depth | 引经据典，文化底蕴 | 文化传承、品牌故事 | ⭐⭐⭐⭐⭐ |
| casual-science | 轻松科普，通俗易懂 | 知识科普、入门教学 | ⭐⭐ |

---

## 1. Dan Koe 风格 (dan-koe)

### 风格特征

**核心理念**: "每句话都必须有价值，每个字都必须有重量"

**语言特点**:
- ✅ 短句为主，长句为辅
- ✅ 去掉一切废话和修饰词
- ✅ 直击要害，不绕弯子
- ✅ 金句频出，适合摘抄分享
- ✅ 强烈的个人观点和态度

**段落结构**:
- 每段2-3句
- 句子之间逻辑紧密
- 段落之间留有呼吸空间
- 适当使用单句成段

**语气**:
- 自信、坚定
- 直接、犀利
- 有态度、有立场
- 避免模棱两可

### 写作公式

#### 1. 观点句式
```
❌ 错误: "我觉得可能也许春茶比较好一些"
✅ 正确: "春茶就是最好的选择。"

❌ 错误: "很多人都说茶文化很重要，我也这么认为"
✅ 正确: "茶文化不是装饰品。是生活方式。"
```

#### 2. 金句创作
```
公式: [现象] ≠ [本质]
示例: "懂茶 ≠ 喝贵茶。懂茶 = 喝对茶。"

公式: [问题] → [真相]
示例: "你以为买的是茶。实际上买的是信息差。"

公式: [对比] + [结论]
示例: "业余的人选茶看价格。专业的人选茶看产地。"
```

#### 3. 段落节奏
```
[开场钩子 - 短句]
[展开论述 - 2-3句]
[结论金句 - 短句]

示例:
"春茶很贵。

不是因为茶叶本身值这个价。
是因为信息不对称让你多付了钱。

明白这一点，你就赢了一半。"
```

### 模板示例

**标题**: 90%的人都在浪费钱买茶

**正文**:
```
你买茶的时候看什么？

价格？
包装？
品牌？

都错了。

真正懂茶的人，只看三个指标：
- 产地
- 采摘时间
- 制作工艺

其他的都是营销。

茶商不会告诉你的真相：
同样的茶，不同的包装，价格能差10倍。

你以为买的是品质。
实际上付的是品牌溢价。

怎么办？

学会看这三个指标。
剩下的，都是噪音。

好茶不贵。
贵的是你的认知成本。

补上这个认知，你就能省下80%的冤枉钱。
```

### 写作清单

**必须做到**:
- ✅ 每句话独立成立，删掉任何一句不影响理解
- ✅ 没有"可能""也许""大概"等模糊词
- ✅ 用词精准，一个词能说清不用两个
- ✅ 金句至少3-5个，可独立传播
- ✅ 强烈的个人观点

**避免使用**:
- ❌ 长句套长句
- ❌ 修饰词堆砌
- ❌ 客套话、废话
- ❌ 模棱两可的表达
- ❌ 被动语态

### 适用模板

| 模板 | 适配度 | 说明 |
|-----|--------|------|
| authoritative | ⭐⭐⭐⭐⭐ | 权威观点 + Dan Koe 短句 = 爆款 |
| comparison | ⭐⭐⭐⭐ | 清晰对比 + 犀利结论 |
| cultural | ⭐⭐⭐ | 需要平衡文化深度和简洁性 |
| practical | ⭐⭐⭐⭐⭐ | 步骤清晰 + 要点明确 |

## 2. 深度文化风格 (cultural-depth)

### 风格特征

**核心理念**: "每一杯茶，都承载着千年文化"

**语言特点**:
- ✅ 引经据典，有文化底蕴
- ✅ 娓娓道来，层层递进
- ✅ 注重意境和韵味
- ✅ 古今结合，承上启下
- ✅ 文字优美，适合朗读

**段落结构**:
- 每段4-5句
- 层层展开，逐步深入
- 注重段落间的过渡
- 营造文化氛围

**语气**:
- 温和、从容
- 深沉、厚重
- 富有情怀
- 引人深思

### 写作公式

#### 1. 引经据典
```
公式: [古籍引用] → [现代解读]
示例:
"《茶经》有云：'茶之为饮，发乎神农氏。'
这不仅是茶的起源，更是中华文明对自然的敬畏。"

公式: [历史典故] → [当代价值]
示例:
"宋徽宗赵佶亲撰《大观茶论》，
不是帝王的闲情逸致，
而是一个时代对美学的极致追求。"
```

#### 2. 意境营造
```
公式: [场景描绘] + [情感升华]
示例:
"清晨，露珠还挂在茶树上。
采茶女的手指轻柔地摘下嫩芽。
这一刻，时间仿佛凝固。
千年的茶文化，就在这一抹绿意中传承。"

公式: [细节描写] + [文化内涵]
示例:
"一个紫砂壶，经年累月的泡养，
表面渐生包浆，温润如玉。
这不是器物的改变，
是时间与茶香的对话。"
```

#### 3. 文化连接
```
公式: [传统] → [传承] → [创新]
示例:
"从陆羽的《茶经》，到现代的茶艺表演，
茶文化跨越千年而不衰。
因为我们始终明白：
真正的传承，不是复制，而是创新。"
```

### 模板示例

**标题**: 一片茶叶里的千年文化密码

**正文**:
```
茶，在中国人的生活中，从来都不只是一杯饮料。

它是陆羽笔下的"南方之嘉木"，
是文人墨客案头的清雅之物，
更是中华文明延续至今的文化符号。

《茶经》开篇即云："茶之为饮，发乎神农氏。"
这一句话，道出了茶与中华文明同龄的历史。
从神农尝百草，到唐宋茶文化鼎盛，
再到明清茶道的精致化，
每一个时代，都赋予了茶新的文化内涵。

宋徽宗在《大观茶论》中，
以帝王之尊，详述点茶之法。
这不是皇权的炫耀，
而是一个时代对极致美学的追求。
点茶七汤法，一汤一韵，一道一味，
在那个时代，喝茶已经上升为一门艺术。

到了今天，我们还需要这些吗？

需要。

因为茶文化传承的，
不是复杂的仪式，
而是对生活的态度：
慢下来，静下心，品味当下。

这是千年前的智慧，
也是现代人最需要的生活哲学。
```

### 写作清单

**必须做到**:
- ✅ 至少引用2-3处古籍或典故
- ✅ 营造文化氛围和意境
- ✅ 古今结合，承上启下
- ✅ 文字优美，有韵味
- ✅ 传递文化价值和情怀

**避免使用**:
- ❌ 生硬堆砌典故
- ❌ 过于白话，失去韵味
- ❌ 只讲古代，不连接现代
- ❌ 说教口吻
- ❌ 华而不实的辞藻

### 适用模板

| 模板 | 适配度 | 说明 |
|-----|--------|------|
| authoritative | ⭐⭐⭐ | 可以加入历史权威 |
| comparison | ⭐⭐ | 文化对比需要平衡 |
| cultural | ⭐⭐⭐⭐⭐ | 完美匹配，最佳选择 |
| practical | ⭐⭐ | 需要平衡实用性和文化性 |

## 3. 轻松科普风格 (casual-science)

### 风格特征

**核心理念**: "让每个人都能听懂，让每个人都感兴趣"

**语言特点**:
- ✅ 通俗易懂，零门槛
- ✅ 多用比喻和生活化例子
- ✅ 有趣味性，不枯燥
- ✅ 亲切自然，像聊天
- ✅ 适当使用emoji增加趣味

**段落结构**:
- 每段3-4句
- 问答式展开
- 多用举例说明
- 逻辑清晰易懂

**语气**:
- 轻松、友好
- 幽默、有趣
- 平易近人
- 鼓励互动

### 写作公式

#### 1. 比喻说明
```
公式: [专业概念] = [生活比喻]
示例:
"茶叶发酵就像做面包：
面粉遇到酵母会膨胀，
茶叶遇到氧气会变色。
原理都是一样的。"

公式: [复杂原理] → [简单类比]
示例:
"为什么绿茶不能用开水泡？
就像炒菜，火太大会糊。
道理都是一样的。"
```

#### 2. 问答式展开
```
公式: [提问] → [回答] → [解释]
示例:
"春茶为什么贵？

不是茶商黑心。
是因为产量少，品质好。

就像草莓，春天的最甜，
但产量少，自然就贵。"

公式: [常见疑问] → [破解真相]
示例:
"很多人问：绿茶和红茶有啥区别？

简单说：
绿茶 = 不发酵（炒青菜）
红茶 = 全发酵（做腌菜）

一个要保鲜，一个要发酵。
完全不同的制作方法。"
```

#### 3. 故事化表达
```
公式: [场景] → [问题] → [解决]
示例:
"上次朋友来家里，
我泡了一杯龙井。

她第一口就皱眉：'怎么这么苦？'

我一看，水温100度。
当场翻车。

后来我才知道，
绿茶最怕开水烫，
80度刚刚好。

从那以后，我家茶桌上
永远放着一个温度计。"
```

### 模板示例

**标题**: 3分钟搞懂：绿茶为什么不能用开水泡

**正文**:
```
很多人泡绿茶都会犯一个错误：
拿开水直接冲。

结果茶汤又苦又涩，
还以为自己买了假茶。

其实不是茶的问题，
是你的泡法不对。

为什么绿茶不能用开水？

道理很简单：
就像炒青菜，火太大会苦。

绿茶里有很多嫩芽，
100度的开水一冲，
叶绿素被破坏，茶多酚过度释放，
当然又苦又涩。

那应该用多少度的水？

答案是：80-85度。

怎么判断80度？

教你一个小技巧：
开水倒进杯子，等1-2分钟，
这时候大概就是80度了。

或者更简单：
买一个带温度计的水壶，
一劳永逸。

试试看，
同样的茶，
用对水温后，
口感能差10倍。

不信？
今天晚上就试试 😊
```

### 写作清单

**必须做到**:
- ✅ 用生活化比喻解释专业概念
- ✅ 多用"你""我"拉近距离
- ✅ 每个知识点都有例子
- ✅ 适当使用emoji增加趣味
- ✅ 鼓励读者动手尝试

**避免使用**:
- ❌ 专业术语不解释
- ❌ 高高在上的说教口吻
- ❌ 长篇大论的理论
- ❌ 没有例子的抽象概念
- ❌ 过于正式的书面语

### 适用模板

| 模板 | 适配度 | 说明 |
|-----|--------|------|
| authoritative | ⭐⭐⭐ | 权威但不严肃 |
| comparison | ⭐⭐⭐⭐ | 轻松对比，易懂 |
| cultural | ⭐⭐ | 需要平衡文化性和趣味性 |
| practical | ⭐⭐⭐⭐⭐ | 完美匹配，最佳选择 |

## 风格混搭技巧

### 混搭原则

**可以混搭**:
- Dan Koe + Authoritative = 权威金句型
- Casual Science + Practical = 轻松实用型
- Cultural Depth + Cultural Template = 深度文化型

**不建议混搭**:
- Dan Koe + Cultural Depth (冲突：简洁 vs 深沉)
- Casual Science + Cultural Depth (冲突：轻松 vs 厚重)

### 混搭示例

**Dan Koe + Authoritative**:
```
"你以为贵的茶就是好茶？

国家一级评茶师告诉你：
好茶不看价格。
看产地、看工艺、看采摘时间。

价格 ≠ 品质。
这是行业秘密。"
```

**Casual Science + Practical**:
```
"泡茶其实很简单！

就像煮方便面：
1. 烧水（80度）
2. 放茶（3克）
3. 倒水（150ml）
4. 等待（2分钟）
5. 倒出（别闷着）

照着做，
保证不踩坑 👍"
```

| 30-50岁 | Dan Koe | 重视效率和价值 |
| 18-30岁 | Casual Science | 喜欢轻松有趣 |

### 根据内容主题

| 主题 | 推荐风格 | 原因 |
|-----|---------|------|
| 茶文化 | Cultural Depth | 需要文化底蕴 |
| 选购指南 | Dan Koe | 需要直接高效 |
| 冲泡技巧 | Casual Science | 需要通俗易懂 |
| 专业知识 | Dan Koe | 需要精准表达 |

### 根据传播目标

| 目标 | 推荐风格 | 策略 |
|-----|---------|------|
| 阅读量最大化 | Casual Science | 降低阅读门槛 |
| 分享率最大化 | Dan Koe | 金句多，适合摘抄 |
| 品牌调性 | Cultural Depth | 建立文化认同 |
| 转化率最大化 | Dan Koe | 直接，有说服力 |

## 风格进阶技巧

### Dan Koe 进阶

1. **金句密度**: 每200字至少1个金句
2. **节奏控制**: 短句 → 中句 → 短句
3. **观点鲜明**: 旗帜鲜明，不留余地
4. **删除测试**: 每句话删掉都可惜

### Cultural Depth 进阶

1. **典故运用**: 至少3个历史典故
2. **意境营造**: 视觉、听觉、嗅觉多感官
3. **时空穿梭**: 古今对照，情景再现
4. **情感共鸣**: 唤起文化认同感

### Casual Science 进阶

1. **比喻库建设**: 准备20+个生活化比喻
2. **互动设计**: 每段加一个互动点
3. **emoji使用**: 适度使用，增加趣味
4. **实验感**: 鼓励读者动手尝试
