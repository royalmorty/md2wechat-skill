# 内容写作模板

## 文章结构模板

### 标准文章结构
```markdown
# 标题

## 引言
[引入话题，设置背景]

## 主体内容
### 第一部分
[详细内容]

### 第二部分
[详细内容]

### 第三部分
[详细内容]

## 结论
[总结观点，行动号召]

## 相关阅读
- [相关链接1]
- [相关链接2]
```

### 列表式文章结构
```markdown
# X个方法/技巧/要点

## 引言
[简要介绍]

## 方法1：标题
[详细说明]

## 方法2：标题
[详细说明]

## 方法3：标题
[详细说明]

## 总结
[核心要点回顾]
```

### 故事式文章结构
```markdown
# 故事标题

## 背景设定
[时间、地点、人物]

## 冲突产生
[问题或挑战]

## 解决过程
[如何克服困难]

## 结果呈现
[最终成果]

## 经验总结
[学到的教训]
```

## 段落模板

### 开场段落
```markdown
在[时间/背景]下，[主题]成为了[群体]关注的焦点。作为[身份/经验]，我发现[核心观点]。今天，我想和大家分享[具体内容]。
```

### 过渡段落
```markdown
说完了[前文内容]，让我们来看看[新内容]。事实上，[新内容]比[前文内容]更加[重要性/复杂性]。
```

### 结论段落
```markdown
通过[方法/经验]，我们不仅[直接结果]，更重要的是[深层意义]。记住[核心要点]，你就能[预期效果]。
```

## 金句模板

### 对比式金句
- "不是[表面现象]，而是[深层本质]"
- "[旧方法]是[缺点]，[新方法]才是[优点]"
- "你以为[错误认知]，实际上[正确认知]"

### 递进式金句
- "[第一步]只是开始，[第二步]才是关键"
- "[基础概念]很重要，但[高级概念]才是核心"
- "[表面现象]容易看到，[深层规律]需要思考"

### 总结式金句
- "[核心要点1] + [核心要点2] = [成功公式]"
- "[方法]不是目的，[目标]才是方向"
- "[过程]可能艰难，但[结果]一定值得"

## 互动引导模板

### 评论引导
```markdown
你对[主题]有什么看法？欢迎在评论区分享你的[经验/观点]。
```

### 分享引导
```markdown
如果你觉得[内容价值]，请分享给需要[相关内容]的朋友。
```

### 收藏引导
```markdown
建议收藏本文，[使用场景]时可以直接参考。
```

## 配图建议模板

### 封面图建议
- **主题相关**：直接体现文章主题
- **情感共鸣**：引发读者情感反应
- **数据可视化**：图表、统计信息
- **场景化**：具体应用场景

### 插图建议
- **步骤说明**：操作流程图
- **对比展示**：前后对比图
- **数据支撑**：图表、数据可视化
- **案例展示**：具体案例图片
