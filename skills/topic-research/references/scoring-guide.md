# 爆款评分算法说明

本文档详细说明 writer score 命令的评分算法和使用方法。

## 评分维度

### 1. 阅读量评分 (30%)

**计算方法**:
```
readScore = min(log10(read_count) * 10, 100) * 0.3
```

**评分标准**:
| 阅读量 | 基础分 | 权重分 |
|-------|--------|--------|
| 1,000 | 30分 | 9分 |
| 10,000 | 40分 | 12分 |
| 100,000 | 50分 | 15分 |
| 1,000,000 | 60分 | 18分 |
| 10,000,000+ | 70分 | 21分 |

**说明**:
- 使用对数计算，避免大数值主导评分
- 阅读量是基础指标，但不是唯一指标
- 10万+是优质内容的门槛

### 2. 互动率评分 (30%)

**计算方法**:
```
engagement_rate = (like_count + share_count + comment_count) / read_count
engagement_score = min(engagement_rate * 100, 30)
```

**评分标准**:
| 互动率 | 水平 | 得分 |
|-------|-----|------|
| < 1% | 偏低 | < 3分 |
| 1-2% | 一般 | 3-6分 |
| 2-5% | 良好 | 6-15分 |
| 5-10% | 优秀 | 15-30分 |
| > 10% | 爆款 | 30分 |

**说明**:
- 互动率是内容质量的核心指标
- 包括点赞、分享、评论三类互动
- 互动率>5%是爆款的重要特征

### 3. 分享率评分 (25%)

**计算方法**:
```
share_rate = share_count / read_count
share_score = min(share_rate * 500, 25)
```

**评分标准**:
| 分享率 | 水平 | 得分 |
|-------|-----|------|
| < 0.5% | 偏低 | < 2.5分 |
| 0.5-1% | 一般 | 2.5-5分 |
| 1-2% | 良好 | 5-10分 |
| 2-5% | 优秀 | 10-25分 |
| > 5% | 爆款 | 25分 |

**说明**:
- 分享是最有价值的互动行为
- 分享意味着用户愿意为内容背书
- 分享率>2%是爆款的关键指标

### 4. 评论率评分 (15%)

**计算方法**:
```
comment_rate = comment_count / read_count
comment_score = min(comment_rate * 300, 15)
```

**评分标准**:
| 评论率 | 水平 | 得分 |
|-------|-----|------|
| < 0.5% | 偏低 | < 1.5分 |
| 0.5-1% | 一般 | 1.5-3分 |
| 1-3% | 良好 | 3-9分 |
| 3-5% | 优秀 | 9-15分 |
| > 5% | 爆款 | 15分 |

**说明**:
- 评论率反映内容的讨论价值
- 高评论率说明内容引发思考
- 评论率/点赞率 > 0.1 说明内容深度好

## 爆款等级

### 等级划分

| 等级 | 分数范围 | 特征 | 传播潜力 |
|-----|---------|------|---------|
| 超级爆款 | 90-100分 | 数据优异，病毒式传播 | 极高 |
| 热门爆款 | 80-89分 | 表现优秀，大量关注 | 高 |
| 优质内容 | 70-79分 | 表现良好，吸引目标受众 | 中上 |
| 潜力内容 | 60-69分 | 有潜力，需优化 | 中 |
| 普通内容 | 50-59分 | 一般，需改进 | 中下 |
| 待优化 | 0-49分 | 表现差，需全面优化 | 低 |

### 等级特征详解

**超级爆款 (90-100分)**
```
典型数据:
- 阅读量: 100万+
- 互动率: 8-15%
- 分享率: 3-8%
- 评论率: 1-3%

特征:
✅ 引发广泛讨论
✅ 跨平台传播
✅ 长尾流量明显
✅ 品牌影响力大

案例:
"90%的人都不知道的XX真相"
"XX背后的惊人秘密"
```

**热门爆款 (80-89分)**
```
典型数据:
- 阅读量: 10万-100万
- 互动率: 5-8%
- 分享率: 2-3%
- 评论率: 0.5-1%

特征:
✅ 目标圈层爆款
✅ 有效传播
✅ 良好互动
✅ 转化率高

案例:
"专家揭秘：XX的5个关键"
"XX大对比：一图看懂"
```

**优质内容 (70-79分)**
```
典型数据:
- 阅读量: 1万-10万
- 互动率: 3-5%
- 分享率: 1-2%
- 评论率: 0.3-0.5%

特征:
✅ 垂直领域认可
✅ 稳定传播
✅ 受众精准
✅ 专业价值高

案例:
"深度解析：XX的原理"
"XX实用指南"
```

## 优化建议生成规则

### 基于互动率的建议

```
if engagement_rate < 0.02:
  建议: "互动率偏低，建议增加互动引导或话题讨论点"

  具体优化方向:
  1. 在文中加入提问环节
  2. 设置互动话题
  3. 引导评论讨论
  4. 增加投票/测试等互动形式
```

### 基于分享率的建议

```
if share_rate < 0.01:
  建议: "分享率偏低，建议增加实用价值或情感共鸣点"

  具体优化方向:
  1. 提供实用干货（工具、方法、清单）
  2. 增强情感共鸣（故事、经历）
  3. 提供社交货币价值（谈资、身份认同）
  4. 优化分享文案和标题
```

### 基于评论率的建议

```
if comment_rate < like_rate * 0.05:
  建议: "评论率偏低，建议增加争议性或思考性内容"

  具体优化方向:
  1. 提出有争议的观点
  2. 抛出开放性问题
  3. 邀请读者分享经验
  4. 设置话题讨论区
```

### 综合优化策略

**低分内容 (< 60分) 优化重点**:
1. 标题优化：加入数字、对比、悬念
2. 开头优化：强化钩子，引发好奇
3. 结构优化：层次分明，逻辑清晰
4. 价值优化：增加实用信息
5. 互动优化：设计互动环节

**中等内容 (60-79分) 优化重点**:
1. 深度优化：增加专业深度
2. 差异化：找到独特角度
3. 传播优化：提升分享动机
4. 数据支撑：增加案例和数据
5. 视觉优化：图表、配图

**优质内容 (80+分) 优化重点**:
1. 时机优化：选择最佳发布时间
2. 渠道优化：多平台分发
3. 持续优化：长尾流量运营
4. 转化优化：引导用户行动
5. 品牌优化：建立品牌认知

## 使用示例

### 示例 1: 评估单个话题

**输入数据**:
```json
{
  "metrics": {
    "read_count": 50000,
    "like_count": 1500,
    "share_count": 500,
    "comment_count": 200
  },
  "topic": "春茶选购指南",
  "platform": "wechat",
  "domain": "tea"
}
```

**命令**:
```bash
echo '...' | ./skills/writer/scripts/writer score
```

**输出结果**:
```json
{
  "score": 75.5,
  "level": "优质内容",
  "engagement_rate": 0.044,
  "share_rate": 0.01,
  "metrics": {...},
  "recommendations": [
    "各项指标表现良好，继续保持！"
  ]
}
```

**分析**:
- 阅读量5万，中等水平
- 互动率4.4%，良好
- 分享率1%，一般
- 综合75.5分，优质内容
- 建议: 提升分享率到2%以上可达80分

### 示例 2: 对比多个话题

**场景**: 从3个候选话题中选择最佳话题

**话题A**: "春茶价格分析"
- 阅读量: 20000
- 互动率: 2%
- 评分: 62分

**话题B**: "春茶选购避坑指南"
- 阅读量: 50000
- 互动率: 4.4%
- 评分: 75分 ⭐

**话题C**: "春茶文化探源"
- 阅读量: 15000
- 互动率: 6%
- 评分: 68分

**建议**: 选择话题B，评分最高，传播潜力最好

### 示例 3: 优化方向分析

**初始数据**:
```
阅读量: 10000
点赞量: 200 (点赞率 2%)
分享量: 50 (分享率 0.5%)
评论量: 30 (评论率 0.3%)
综合评分: 58分
```

**优化建议**:
1. ⚠️ 分享率偏低（0.5% < 1%）→ 增加实用价值
2. ⚠️ 评论率偏低（0.3% < 点赞率*0.05=0.1%）→ 增加讨论点
3. 💡 阅读量基础尚可，重点优化互动

**优化后预期**:
```
如果分享率提升到1.5%:
- 分享评分: +2.5分
- 综合评分: 60.5分 → 进入"潜力内容"级别

如果互动率提升到5%:
- 互动评分: +9分
- 综合评分: 67分 → 接近"优质内容"
```

## 评分调整因子

### 领域调整

不同领域的数据基准不同，需要调整评分标准：

| 领域 | 阅读量基准 | 互动率基准 | 调整系数 |
|-----|----------|-----------|---------|
| 茶文化 | 中等 | 较高 | 1.0 |
| 科技 | 较高 | 中等 | 1.1 |
| 生活方式 | 高 | 高 | 1.2 |

### 平台调整

不同平台的互动特征不同：

| 平台 | 特点 | 调整建议 |
|-----|------|---------|
| 微信公众号 | 闭环传播 | 重视分享率 |
| 微博 | 开放传播 | 重视转发率 |
| 知乎 | 深度内容 | 重视收藏率 |

### 时间调整

内容时效性影响评分：

| 时效性 | 适用内容 | 评分调整 |
|-------|---------|---------|
| 即时热点 | 新闻、事件 | +10% |
| 季节性 | 春茶、节日 | +5% |
| 常青内容 | 知识、技能 | 标准 |

## 注意事项

1. **数据真实性**: 评分基于真实数据，避免刷量
2. **相对性**: 评分是相对的，不同领域标准不同
3. **动态性**: 评分会随时间变化，及时监测
4. **综合性**: 不要只看单一指标，要综合评估
5. **目标导向**: 根据目标选择优化方向
