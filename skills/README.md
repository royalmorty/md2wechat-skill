# 技能使用指南

## 技能概览

微信公众号创作技能套装包含6个专业技能，覆盖从选题到分析的完整内容创作流程：

### 1. topic-research（选题研究）
- **类型**：自动触发
- **功能**：热点分析、竞品研究、受众调研
- **触发词**："选题"、"热点"、"趋势"、"调研"
- **输出**：选题报告 + 3-5个推荐选题

### 2. content-writing（内容创作）
- **类型**：手动触发
- **用法**：`/content-writing [文章主题]`
- **功能**：文章撰写、结构优化、内容润色
- **输出**：完整文章 + 创作说明

### 3. visual-design（视觉设计）
- **类型**：自动触发
- **功能**：配图建议、封面设计、版式优化
- **触发词**："配图"、"封面"、"视觉"、"设计"
- **输出**：设计建议 + 具体方案

### 4. seo-optimization（SEO优化）
- **类型**：手动触发
- **用法**：`/seo-optimization [文章内容]`
- **功能**：关键词研究、标题优化、搜索排名提升
- **输出**：优化报告 + 具体建议

### 5. publishing-strategy（发布策略）
- **类型**：自动触发
- **功能**：发布时间、标签优化、互动策略
- **触发词**："发布"、"策略"、"推广"、"时机"
- **输出**：策略报告 + 执行方案

### 6. content-analysis（内容分析）
- **类型**：手动触发
- **用法**：`/content-analysis [分析目标]`
- **功能**：数据洞察、效果评估、优化建议
- **输出**：分析报告 + 改进方案

## 使用流程

### 完整创作流程
```
1. 选题阶段 → topic-research（自动）
   ↓
2. 创作阶段 → content-writing（手动）
   ↓  
3. 设计阶段 → visual-design（自动）
   ↓
4. 优化阶段 → seo-optimization（手动）
   ↓
5. 发布阶段 → publishing-strategy（自动）
   ↓
6. 分析阶段 → content-analysis（手动）
```

### 快速创作流程
```
1. 直接创作 → content-writing（手动）
   ↓
2. 视觉优化 → visual-design（自动）
   ↓
3. SEO优化 → seo-optimization（手动）
```

### 优化改进流程
```
1. 问题诊断 → content-analysis（手动）
   ↓
2. 选题调整 → topic-research（自动）
   ↓
3. 内容重写 → content-writing（手动）
   ↓
4. 重新优化 → seo-optimization（手动）
```

## 使用技巧

### 技能组合使用
- **新账号启动**：topic-research → content-writing → visual-design → publishing-strategy
- **热点追踪**：topic-research → content-writing → publishing-strategy（快速响应）
- **内容优化**：content-analysis → seo-optimization → content-writing（改进版）
- **系列策划**：topic-research → content-writing × N → content-analysis

### 触发词优化
- 使用具体、明确的触发词
- 结合上下文语境
- 避免歧义和模糊表达
- 可以同时触发多个相关技能

### 参数传递技巧
- 手动技能支持详细的参数输入
- 可以传递文章草稿、关键词、分析目标等
- 参数越具体，输出越精准
- 支持多轮对话和参数调整

## 最佳实践

### 内容创作最佳实践
1. **选题精准**：使用topic-research确保选题质量
2. **内容优质**：依赖content-writing的专业写作能力
3. **视觉统一**：visual-design确保品牌形象一致性
4. **SEO到位**：seo-optimization提升搜索排名
5. **发布科学**：publishing-strategy最大化传播效果
6. **持续优化**：content-analysis持续改进内容策略

### 效率提升建议
- 建立标准化的创作流程
- 积累高质量的内容模板
- 定期分析内容表现数据
- 建立选题和素材库
- 培养团队协作机制

### 质量控制要点
- 每篇文章都经过完整的优化流程
- 建立内容质量评估标准
- 定期回顾和更新技能使用策略
- 关注平台规则变化和趋势更新
- 保持内容的原创性和价值性

## 注意事项

### 技能使用限制
- 自动技能基于关键词触发，可能不够精准
- 手动技能需要明确的参数输入
- 某些技能需要外部数据支持
- 技能输出需要人工判断和调整

### 内容合规要求
- 所有内容必须符合微信平台规范
- 避免敏感话题和违规内容
- 重视版权保护和素材合法性
- 保持内容的客观性和专业性

### 数据隐私保护
- 保护用户数据和隐私信息
- 遵守相关的数据保护法规
- 合理使用用户生成内容
- 建立数据安全保护机制

## 故障排除

### 技能未触发
- 检查触发词是否准确
- 确认技能配置是否正确
- 查看是否有权限限制
- 尝试重新启动对话

### 输出不符合预期
- 检查输入参数是否完整
- 确认上下文语境是否清晰
- 尝试调整技能调用方式
- 联系技术支持获取帮助

### 技能冲突处理
- 多个技能同时触发时，选择最相关的
- 手动技能优先级高于自动技能
- 可以明确指定需要的技能
- 建立技能使用的优先级规则

通过合理使用这些技能，你可以大幅提升微信公众号内容创作的效率和质量，建立系统化的内容创作体系。
